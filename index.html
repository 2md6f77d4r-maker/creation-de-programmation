🚀 Mise à jour du logiciel

Version V3.18 — Auto-save renforcée

Le logiciel conserve maintenant automatiquement votre programmation dans le navigateur utilisé.

💾
Votre planning, vos modifications manuelles, les services et les paramètres sont mémorisés automatiquement.
⏱️
Une sauvegarde se fait immédiatement après les actions importantes et une vérification automatique se fait aussi toutes les 5 secondes.
🟢
Un indicateur affiche la dernière sauvegarde connue pour vérifier rapidement que tout est bien mémorisé.
🌐
La sauvegarde reste liée à ce navigateur et à cet appareil. Pour retrouver les informations, utilisez le même navigateur.
🔒
Aucune donnée n’est envoyée sur un serveur: la sauvegarde reste locale sur votre appareil.
🧹
Le bouton Réinitialiser supprime le planning affiché et efface la sauvegarde automatique.
⚠️
Si vous videz les données du navigateur, utilisez un mode privé ou changez de navigateur, la sauvegarde peut disparaître.

Conseil : pour garder une copie externe, exportez aussi régulièrement le planning en Excel ou en PDF.
